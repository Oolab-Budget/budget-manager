Budget Manager - Quick Reference Guide

Common Tasks - Step by Step

Adding Items to Budget

  1. Go to Budget tab
  2. Select Category from dropdown
  3. Click Add to Budget next to item
  4. Edit quantity/price if needed

Creating a New Category

  1. Go to Categories tab
  2. Click Add New Category
  3. Enter Category Key (lowercase, no spaces)
  4. Enter Name (English and Spanish)
  5. Click Add Category

Adding New Item to Database

  1. Go to Manage Items tab
  2. Select category
  3. Click Add New Item
  4. Fill in all details
  5. Check Recurring Weekly if needed
  6. Click Add Item

Saving a Budget

  1. Go to Budget tab
  2. Review all items
  3. Click Save Budget button
  4. Enter label (optional)
  5. Check Partial Payment if needed
  6. Click Save

Creating Can/Mex Request

  1. Go to Can/Mex tab
  2. Select category or search for items
  3. Click Add next to items
  4. Set quantities
  5. Add tracking numbers (optional)
  6. Click Generate Request

Adding UPS Tracking

  1. Go to Can/Mex or Mex/Can tab
  2. Find request in history
  3. Click Add Tracking Number
  4. Enter UPS tracking number
  5. Status updates automatically

Generating Report

  1. Go to Reports tab
  2. Select report type (Budget or Canada)
  3. Select date range
  4. Select item filter (if needed)
  5. Report displays automatically

Backing Up Data

  1. Go to Settings tab
  2. Scroll to Backup & Restore
  3. Click Backup All Data
  4. Choose save location
  5. File saved with timestamp

Restoring from Backup

  1. Go to Settings tab
  2. Scroll to Backup & Restore
  3. Click Choose File
  4. Select backup JSON file
  5. Click Restore Backup
  6. Wait for confirmation

Configuring Email (EmailJS)

  1. Go to Settings tab
  2. Scroll to Email Configuration
  3. Enter Public Key
  4. Enter Service ID
  5. Enter Template ID
  6. Enter Recipient Email
  7. Click Save Email Settings

Setting Up UPS Tracking

  1. Go to Settings tab
  2. Scroll to UPS Tracking
  3. Enter Client ID
  4. Enter Client Secret
  5. Click Save UPS Settings

Marking Item as Recurring

  1. Go to Manage Items tab
  2. Find item
  3. Click Edit
  4. Check Recurring Weekly or Recurring Monthly
  5. If monthly, set Monthly Day
  6. Click Save Changes

Adding Receipt to Item

  1. Go to Budget tab
  2. Find item in budget table
  3. Click Add Receipt button
  4. Select image file
  5. Receipt is saved

Viewing Saved Budget

  1. Go to Reports tab
  2. Select Saved Budgets report type
  3. Click View Details on any budget
  4. See full breakdown
  5. Add receipts if needed

Filtering Canada Reports by Item

  1. Go to Reports tab
  2. Click Canada Reports button
  3. Select Request History report type
  4. Select date range
  5. Select item from Filter by Item dropdown
  6. See item-focused table with totals

Dismissing Backorder

  1. Go to Can/Mex tab
  2. Scroll to Backorders Summary
  3. Find backorder group
  4. Click Dismiss button
  5. Backorder is hidden

Switching Language

  • Click ES / EN button in top-left corner
  • All text switches immediately

Exporting to Excel

  1. Go to Budget tab
  2. Click Export to Excel button
  3. File downloads automatically

Important Notes

  • Recurring Items: Add automatically every Monday
  • Manually Removed Items: Won't reappear until next week
  • Backups: Created automatically, saved to desktop
  • Tracking Updates: Check every hour automatically
  • Partial Payments: Can save budgets not fully paid
  • Item Notes: Available in Mex/Can requests only

Quick Tips

  • Use Search to find items quickly
  • Save Budget frequently to prevent data loss
  • Add Receipts immediately after purchase
  • Check Tracking regularly for delays
  • Backup Data before major changes
  • Use Filters in reports to focus on specific data

Common Shortcuts

Shortcut Action
Ctrl+Shift+R Hard refresh
F12 Open console (for troubleshooting)
Ctrl+F Search page

Need Help?

  1. Check USER_MANUAL.html for detailed instructions
  2. Check browser console (F12) for errors
  3. Try hard refresh (Ctrl+Shift+R)
  4. Restore from backup if data is lost
  5. Contact system administrator
